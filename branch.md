# Temel Branch Islemleri
-----------------------
### Branch olusturma:
`git branch <branch_ismi>`
  
### Branch degistirme:     
`git checkout <branch_ismi>`

### Branch kaldirma:      
`git branch -d <branch_ismi>`

### Branch ismi degistirme:    
`git branch -m <eski_branch_ismi> <yeni_branch_ismi>`

### Eski branchi kaldirma:    
`git push origin : <eski_branch_ismi>`

### Yeni branhi sunucuya aktarma:
`git push origin <branch_ismi>`   
`git push --set-upstream origin <branch_ismi>`
